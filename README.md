# Storyteller App

A fun browser app that generates a random short story based on your chosen genre.

## What it does

Click one of three buttons — Scary, Funny, or Adventure — and the app picks a random story from that category and displays it on the page.

## Files

- `index.html` — three genre buttons and a result display area
- `styles.css` — styling
- `script.js` — story arrays and random selection logic

## Technologies

HTML, CSS, JavaScript
